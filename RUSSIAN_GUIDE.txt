ЕСЛИ У ВАС ОШИБКА С КИРИЛЛИЦЕЙ - ЧИТАЙТЕ ЭТО!

Проблема: Windows CMD не любит русские буквы в скриптах.
Решение: Все скрипты теперь на английском языке!


═══════════════════════════════════════════════════════════════════════
  ШАГ 1: ЧТО УСТАНОВИТЬ (10 минут)
═══════════════════════════════════════════════════════════════════════

1. Git
   Ссылка: https://git-scm.com/download/win
   Нажимайте "Next" везде
   ПЕРЕЗАГРУЗИТЕ компьютер после установки!

2. .NET 8.0 SDK
   Ссылка: https://dotnet.microsoft.com/download/dotnet/8.0
   Выберите "SDK x64" для Windows
   
3. GitHub аккаунт
   Ссылка: https://github.com/signup
   Запомните ваш username!

4. Railway аккаунт
   Ссылка: https://railway.app
   Войдите через GitHub


═══════════════════════════════════════════════════════════════════════
  ШАГ 2: ЗАПУСК (20 минут)
═══════════════════════════════════════════════════════════════════════

1. Найдите файл MASTER_SETUP.bat

2. ПРАВОЙ кнопкой мыши → Запуск от имени администратора

3. Выберите [1] - Full setup

4. Введите данные когда попросит:
   - GitHub username (ваш логин на GitHub)
   - Repository name (например: hardcore-messenger)

5. Когда скрипт попросит - создайте репозиторий на GitHub:
   - Откройте https://github.com/new
   - Repository name: hardcore-messenger
   - Выберите Public
   - НЕ ставьте галочки!
   - Create repository

6. Нажмите любую клавишу в скрипте - код загрузится!


═══════════════════════════════════════════════════════════════════════
  ШАГ 3: RAILWAY (5 минут)
═══════════════════════════════════════════════════════════════════════

1. Откройте https://railway.app

2. New Project → Deploy from GitHub repo

3. Найдите hardcore-messenger (ваш репозиторий)

4. Подождите 2-3 минуты пока Railway развернёт сервер

5. Settings → Networking → Generate Domain

6. СКОПИРУЙТЕ URL! Например:
   hardcore-messenger-production.up.railway.app


═══════════════════════════════════════════════════════════════════════
  ШАГ 4: EXE ФАЙЛ (автоматически)
═══════════════════════════════════════════════════════════════════════

Скрипт автоматически создаст EXE файл!

Найдёте его в папке Distribution\


═══════════════════════════════════════════════════════════════════════
  ШАГ 5: НАСТРОЙКА (1 минута)
═══════════════════════════════════════════════════════════════════════

1. Откройте Distribution\ClientConfig.txt

2. Замените:
   ws://YOUR_RAILWAY_URL_HERE
   
   На ваш Railway URL:
   ws://hardcore-messenger-production.up.railway.app

3. Сохраните (Ctrl+S)


═══════════════════════════════════════════════════════════════════════
  ШАГ 6: ТЕСТ (2 минуты)
═══════════════════════════════════════════════════════════════════════

1. Откройте Distribution\HardcoreMessenger.exe

2. Если Windows ругается:
   - Подробнее → Выполнить в любом случае

3. Введите ваше имя

4. Нажмите "Подключиться"

5. РАБОТАЕТ! 🎉


═══════════════════════════════════════════════════════════════════════
  ЧТО ДАЛЬШЕ?

3. Дайте ссылку друзьям!


═══════════════════════════════════════════════════════════════════════
  ЧАСТЫЕ ПРОБЛЕМЫ
═══════════════════════════════════════════════════════════════════════

Проблема: "Git is not recognized"
Решение: Перезагрузите компьютер после установки Git

Проблема: "dotnet is not recognized"  
Решение: Установите .NET 8.0 SDK (не Runtime!)

Проблема: Клиент не подключается
Решение: 
  1. Проверьте ClientConfig.txt
  2. URL должен начинаться с ws://
  3. Проверьте что Railway сервер запущен

Проблема: Кириллица в скриптах
Решение: Теперь все скрипты на английском!
